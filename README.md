# @shade-protocol/contracts

Smart contracts for Shade Protocol, built with [Foundry](https://book.getfoundry.sh/).

## Contracts

| Contract | Description |
|---|---|
| `ShadePool.sol` | Entry point — `shield()` and `transact()` |
| `Commitments.sol` | Binary Poseidon Merkle tree (depth 16) |
| `TokenGuard.sol` | cBTC/WcBTC wrapping and unwrapping |
| `GeneratedVerifier.sol` | Groth16 proof verifier (generated by snarkjs) |
| `Types.sol` | Shared data structures |

## Target Chain

Citrea mainnet (chain ID 4114) — a Bitcoin L2 with full EVM and bn128 precompile support.

## Build

```bash
forge build
```

## Test

```bash
forge test -vvv
```

10 tests covering shield, transact, double-spend prevention, invalid proofs, and edge cases.

## Deploy

```bash
forge script script/Deploy.s.sol --rpc-url https://rpc.citreascan.com --broadcast
```

Requires `DEPLOYER_PRIVATE_KEY`, `WCBTC_ADDRESS`, `VERIFIER_ADDRESS` environment variables.
